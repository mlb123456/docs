# 7.1自定义指令

  • 7.1.1 自定义全局指令
  1. 注册全局指令 所谓全局指令，就是可以在多个Vue实例中使用的指令。 注册全局指令时，指令名不要用大写字母，且不要加v-,但是使用时一定要加前缀v-。
Vue.directive('指令名', {
    自定义对象
});
Vue.directive('指令名', {
    bind(el, binding, vnode) {},
    inserted(el, binding, vnode) {},
    update(el, binding, vnode, oldVnode) {},
    componentUpdated(el, binding, vnode) {},
    unbind(el, binding, vnode) {},
});

定义对象为钩子函数，钩子函数主要有5个,就是上面代码中示例的五个。 以上钩子函数可以根据需要进行定义，钩子函数根据需要可以有参数，也可以没有参数。

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <script src="https://cdn.jsdelivr.net/npm/vue@2.7.10/dist/vue.js"></script>
    <title>Document</title>
</head>

<body>
    <div id="app">
        <h3 v-my-dir="msg">{{msg}}</h3>
        <button @click="change">更新数据</button>
        <button @click="jiebang">解绑</button>
    </div>
</body>
<script>
    let vm = new Vue({
        el: '#app',
        data: {
            msg: 'hello world'
        },
        methods: {
            change() {
                this.msg = '欢迎到来'
            },
            jiebang() {
                // 实例与DOM元素解绑后，vm实例中的指令自然也与DOM元素解绑了
                vm.$destroy();
            }
        }
    })
    Vue.directive('my-dir', {
        bind() {
            // 常用
            // 此例的指令绑定到h3时就执行
            alert('指令第一次绑定到元素上时调用，只调用一次，一般执行初始化操作');
        },
        inserted() {
            // 被绑定元素此例为h3，h3插入div时执行
            alert('被绑定元素插入DOM中调用');
        },
        update() {
            alert('被绑定元素所在模板更新时调用');
        },
        componentUpdated() {
            alert('被绑定元素所在模板完成一次更新周期时调用，相当于已经完成更新时调用');
        },
        unbind() {
            alert('指令与元素解绑时调用，只调用一次');
        },
    });


</script>

</html>

            console.log(binding.expression);
            console.log('arg-----' + binding.arg);
            el.innerHTML = binding.value.charAt(0).toUpperCase() + binding.value.slice(1);
        },
    });
    let vm = new Vue({

全局指令还有一个特点该指令可以在所有的Vue实例中使用：

<div id="app2">
        <h1 v-red-text="msg"></h1>
</div>

<script>
    let vm2 = new Vue({
        el: '#app2',
        data: {
            msg: 'gogogo'
        }
    })
</script>
  1. 钩子函数简写

在很多时候，可能想在bind和updata时触发相同行为，而不关心其他钩子函数。比如上面的自定义指令可以写成这样：

Vue.directive('red-text', function (el, binding){
    el.innerHTML = binding.value.charAt(0).toUpperCase()+binding.value.slice(1);
    el.style.color = 'red';
});

即第二个参数直接传入一个函数，这个函数直接相当于bind和update钩子函数，也就是在刚绑定的元素上会执行，在被绑定元素更新时也会执行。

  • 7.1.2 自定义局部指令
    在某个Vue实例内部定义，即通过Vue实例选项directives:{}定义的指令为局部指令；
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <script src="https://cdn.jsdelivr.net/npm/vue@2.7.10/dist/vue.js"></script>
    <title>Document</title>
</head>

<body>
    <div id="app">
        被绑定的元素自动获取焦点：<input type="text" v-myfocus>
    </div>
    <div id="app2">
        被绑定的元素自动获取焦点2：<input type="text" v-myfocus>
    </div>
</body>
<script>
    let vm = new Vue({
        el: '#app',
        data: {
            msg: 'welcome to here'
        },
        directives: {
            'myfocus': { //指令名不能用大写字母
                inserted(el) {
                    el.focus();
                },
            }
        },
    })

    // 其他Vue实例不能使用自定义局部指令
    var vm2 = new Vue({
        el: '#app2',
        data: {
            msg: 'welcome to china  '
        }
    })
</script>

</html>

    Vue在插入、更新、移除DOM时，提供多种不同方式的应用过渡效果。
